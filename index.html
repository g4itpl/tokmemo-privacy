톡메모 정리함 개인정보처리방침

시행일: 2026년 7월 10일

톡메모 정리함(이하 "앱")은 카카오톡에서 사용자가 남긴 메모와 사용자가 선택한 다른 채팅의 유용한 정보를 기기 안에서 정리하는 독립 앱입니다. 카카오 및 카카오톡과 제휴하거나 카카오가 제공하는 공식 앱이 아닙니다.

1. 접근하는 정보

  • 알림 접근을 허용한 경우: 카카오톡 알림의 방 이름, 보낸 사람 또는 알림 제목, 메시지 내용, 알림 시각
  • 사용자가 직접 공유하거나 가져온 경우: 텍스트, 링크, 사진, 동영상, 문서와 파일 정보
  • 앱 설정: 수집할 방 이름, 필터, 차단 목록, 테마와 같은 사용 환경

2. 이용 목적과 처리 방식

정보는 메모 분류, 검색, 일정·링크·장소·파일 실행, 중복 제거와 사용자 요청에 따른 백업·복원에만 사용됩니다. 알림 접근 권한으로 들어오는 정보 중 앱의 저장 조건에 맞지 않는 내용은 메모로 저장하지 않습니다.

3. 외부 전송과 제3자 제공

앱에는 인터넷 권한이 없으며 개발자가 운영하는 서버, 광고 서비스 또는 분석 서비스로 메모를 전송하지 않습니다. 사용자가 일정 등록, 링크 열기, 지도 검색 또는 첨부파일 열기를 직접 선택하면 해당 내용이 사용자가 선택한 외부 앱으로 전달되며 이후 처리는 그 앱의 정책을 따릅니다.

4. 저장 위치와 보유 기간

메모와 앱으로 복사된 첨부파일은 Android 앱 전용 저장공간에 보관됩니다. 사용자가 개별 항목이나 전체 데이터를 삭제하거나 앱을 제거할 때까지 보관됩니다. 전체 삭제 기능은 메모와 앱 내부 첨부파일을 함께 삭제합니다.

5. 암호화 백업

사용자가 백업을 요청할 때만 선택한 위치에 .tokmemo 파일을 만듭니다. 백업은 사용자가 정한 암호로 AES-GCM 방식으로 암호화되며 암호는 앱에 저장되지 않습니다. 암호를 잊으면 백업을 복원할 수 없습니다.

6. 민감정보 보호

명확한 일회용 인증번호, OTP, 비밀번호 또는 복구 코드로 판단되는 메시지는 저장 단계에서 제외합니다. 사용자는 휴대폰 잠금 확인을 이용한 앱 잠금을 선택할 수 있습니다.

7. 사용자의 선택과 삭제

알림 접근은 Android 설정에서 언제든 해제할 수 있습니다. 앱의 관리 화면에서 개별 메모 삭제, 후보 차단, 전체 데이터 삭제, 암호화 백업과 복원을 이용할 수 있습니다.

8. 아동의 개인정보

앱은 만 13세 미만 아동을 대상으로 설계되지 않았으며 아동의 개인정보를 의도적으로 수집하지 않습니다.

9. 방침 변경

앱의 데이터 처리 방식이 달라지면 시행 전에 이 페이지와 앱 안의 안내를 갱신합니다.

10. 문의

개인정보 관련 문의는 공개 문의 페이지를 이용할 수 있습니다. 공개 문의에는 실제 대화 내용, 백업 암호 또는 민감정보를 작성하지 마세요.
